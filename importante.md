# Comandos a tener en cuenta

Algunpos comandos importantes para manejar Cassandra son:

- Ver el estado de los nodos:

`nodetool status`

- Iniciar, parar o resetear el servicio de Cassandra:

`systemctl start/stop/restart cassandra`
